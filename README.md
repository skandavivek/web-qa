## **_To run:_**

```bash
# create and activate python environment
$ python -m venv .venv
$ .\.venv\Scripts\activate

# change directory
$ cd open-ai-app/

# install project requirements
$ pip install -r requirements.txt

# start application
$ flask run

# NOTE: All app configurations are inside .env file
```
